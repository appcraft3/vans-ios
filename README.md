# VanGo

A trust-first, invite-only community for van lifers. Join real-world events, date other nomads, and make activity-based connections. Includes trusted builder help, monetized with VanGo Pro.

## Features

- **Invite-Only Access** — Waitlist + admin review to keep the community trusted
- **Events** — Create and join van life events (hiking, surfing, yoga, stargazing, and more)
- **Matching** — Express interest at events; mutual interest creates a match
- **Messaging** — Real-time 1-on-1 and event group chat
- **Stories** — Share 24-hour photo stories on the Explore map
- **Trust System** — Reputation badges earned through events and reviews
- **Builder Network** — Find trusted builders for van conversions and repairs
- **VanGo Pro** — Priority review, Verified+ badge, visibility boost, and priority support

## Tech Stack

### iOS

- **Swift** / **SwiftUI** + **UIKit** (MVVM-C architecture)
- **Firebase iOS SDK** — Auth, Firestore, Cloud Functions, Remote Config
- **Google Sign-In** / **Sign in with Apple**
- **RevenueCat** — Subscriptions & entitlements
- **MapKit** — Event discovery map with dark styling
- **Kingfisher** — Image loading & caching
- **Combine** — Reactive data binding
- **Swift Package Manager**

### Backend

- **Firebase Cloud Functions** (Node.js)
- **Cloud Firestore** — Real-time database
- **Firebase Authentication** — Google & Apple OAuth
- **Firebase Remote Config** — Feature flags & force updates
